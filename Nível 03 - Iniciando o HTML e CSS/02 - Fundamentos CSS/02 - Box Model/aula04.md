# Display Inline
  1. Ocupa apenas o espaço do conteúdo do elemento.
  2. Elementos poderão ficar `em linha`
  3. `width` e `height` não se aplicam
  4. Aplicamos apenas os valores horizontais de `margin`, `padding` e `border`.

  > elementos
  `<a>, <span>, <strong>, <em>`

  ```html
    <span>inline</span>
    <span>inline</span>
  ```

  ```css
    div: {
      border: 10px solid;

      width: 200px;
      height: 200px;
      
      margin: 20px; 
      padding: 20px;
    }
  ```
