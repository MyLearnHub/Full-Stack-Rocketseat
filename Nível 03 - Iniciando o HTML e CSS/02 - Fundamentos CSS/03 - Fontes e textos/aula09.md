# Web Fonts
  ```html
    <link rel="preconnect" href="https://fonts.googleapis.com">
    <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>

    <link href="https://fonts.googleapis.com/css2?family=Madimi+One&family=Roboto:ital,wght@0,100..900;1,100..900&display=swap" rel="stylesheet">
  ```

  ```css
    @import url('https://fonts.googleapis.com/css2?family=Madimi+One&family=Roboto:ital,wght@0,100..900;1,100..900&display=swap');

    h1 {
      font-family: "Madimi One", sans-serif;
    }

    p {
      font-family: 'Courier New', Courier, monospace;
      font-family: 'Roboto', sans-serif;
      font-weight: 300;
    }
  ```
