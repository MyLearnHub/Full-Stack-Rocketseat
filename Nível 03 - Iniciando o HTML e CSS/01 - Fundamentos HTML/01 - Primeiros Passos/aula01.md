# O que é o HTML?

## HTML
  Hypertext (Hipertexto)
  Markup (Marcação)
  Language (Linguagem)

  ---
  
  ### O que é o HTML?
  - É linguagem de programação?
  - `tags` (marcação)
  - Links para outros textos
  - Imagens, Sons e Vídeos
  - Arquivo com a extensão `.html`

  ### Exemplo
  ```html
    <a href="https://google.com" target="_blank">google</a>
  ```
