# Anatomia das Tags
  - Abertura de tag
  - Conteúdo
  - Fechamento de tag
  - Elementos
  - Atributos

## Exemplo
  `<h1>Título</h1>` - Título

  ```html
    <h1 id="title">Título</h1>
  ```
## Elementos vazios
  - Não possuem conteúdo
  - Poderão conter atributos
  `<img src="" alt="" />` - Imagem
  `<br/>` (Break Role) - Quebra de Linha

  ### Exemplos
  ```html
    <img src="" alt="" />
    <br/>
  ```
